# Integration
Apps can easily integrate GreedyGame SDKX by adding the dependency to `package.json`
#### Add the dependency to your app
``` gradle
    npm i @greedygame/react-native-sdkx
```

or

```gradle
    yarn add @greedygame/react-native-sdkx
```
